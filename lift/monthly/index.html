FAQ

How many variations do you create each month?

Just the one unless we’ve agreed otherwise, but it’s the one I believe is the best to increase your conversions.

How many revisions do I get on each variation?

You may have up to 2 revisions on the copy and 2 revisions on the design.

How do we tell if the new variation worked?

We run what’s called an A/B test between the current landing page and the new variation. When we run the test, the traffic is split evenly between the old and new landing pages and any conversions are recorded.

So what happens if the new variation doesn’t increase the conversion rate?

It happens—we don’t get it right every time. When it does we pick ourselves up, dust ourselves off and get back on the horse. I then analyse the results of the A/B test and create a new variation.

Can the pages be viewed on phones and tablets?

‘Responsive design’ is standard so that your pages look great on tablets and mobiles.

Is coding included?

Each design is carefully crafted pixel by pixel into clean HTML, CSS and Javascript code. We offer responsive design as standard so your pages look great on tablet and mobile devices.

Can you integrate WordPress?

WordPress integration is not included, but if this is a requirement please let me know and I’ll provide you with a custom quote.

Do you offer hosting for the landing page?

Yes, all the pages are hosted on the fast and reliable Rackspace CDN network at no extra cost. You can also use a custom domain or sub domain with your landing pages.
